# 7 Days of Code

Este projeto faz parte de um desafio de 7 dias, no qual será implementada uma nova funcionalidade a cada dia.

## Funcionalidades

- Dia 1: Implementação do Cabeçalho (Navbar)

    Criamos a estrutura do cabeçalho da página, conhecido como Navbar, que inclui o logotipo da empresa, um menu de navegação com links para as principais páginas (Home, Produtos, Recursos e Sobre Nós) e dois botões de ação no canto direito (Login e Cadastre-se). Utilizamos HTML para estruturar o conteúdo e CSS para estilizar os elementos, garantindo um design moderno e responsivo.
